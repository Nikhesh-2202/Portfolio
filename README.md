# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, built completely from scratch using **HTML** and **CSS**, hosted on **GitHub**, and deployed with **Vercel**.

Instead of relying on templates or website builders, I wanted to create every section myself. It gave me complete control over the design, layout, and user experience while helping me strengthen my understanding of frontend development fundamentals. Every component—from the navigation to the responsive layout—was built with handwritten code.

## Technical Highlights

While building this project, I focused on writing clean and organized code rather than simply making the page look good.

Some implementation details include:

* Responsive layouts using modern CSS techniques
* Consistent spacing, typography, and visual hierarchy
* Semantic HTML for improved accessibility and maintainability
* Mobile-first thinking to ensure a smooth experience on different screen sizes
* Lightweight structure for faster loading and better performance
* Reusable CSS classes to keep the stylesheet clean and easy to manage
